Examen Parcial


Se pide realizar una página web. Debe contener al menos las siguientes rutas.


  • / . Donde habrá un campo de texto para introducir un numero de teléfono y un botón de enviar
  • /country/el_pais . Donde se mostará información de el_pais
  • /city/la_ciudad . Donde se mostrará información sobre la_ciudad


Explicación Rutas


/


Se monstrará un input de texto para introducir un teléfono y un botón para enviar. Al enviar el formulario se cargará la misma ruta / indicando los siguientes datos:


  • Número de teléfono.
  • País (Esto será un link que abrirá /country/pais)

Si no se ha introducido el número de teléfono se mostrará solo el formulario.

Si el número de teléfono no es válido se mostrará el formulario y un mensaje de error indicando: “Número de teléfóno incorrecto”.


/country/el_pais


Se mostrará la siguiente información


  • Nombre del país
  • Capital (Esto será un link que abrirá /city/capital)


/city/la_ciudad


Se mostrará la siguiente información


  • Nombre de la ciudad
  • País (Esto será un link que abrirá /country/pais)
  • Temperatura

Además todas las páginas tendrán una cabecera que contenga un enlace a / y un footer que ponga: “Copyright Nebrija 2025”

La cabecera estará en la parte superior de la página y el footer en la parte inferior.


Puntuación


  • ruta / - 2.5 puntos
  • ruta /country/el_pais - 2.5 puntos
  • ruta /city/la_ciudad - 2.5 puntos
  • Estrucutura, calidad y claridad del código - 2.5 puntos
